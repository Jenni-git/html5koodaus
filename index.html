Asiaa web-standardeista

Tärkeimmät web-standardit ovat HTML, XML, CSS ja JavaScript. Näiden tekniikoiden avulla tehdään web-sivuja, joita voidaan käyttää mitä erilaisimmilla päätelaitteilla ja ohjelmistoilla.

Seuraa yllä olevia painikkeita.


HTML ja mihin sitä käytetään?

HTML on tehty universaaliksi julkaisukieleksi, jota tietokoneet ja niissä käytetyt ohjelmat ymmärtävät. HTML:n avulla sisällöntuottajat saavat julkaistua WWW:ssä dokumentteja verkossa luettavaksi. HTML-komentojen avulla määritellään dokumentin rakenne. Sisältö voi olla tekstiä, taulukoita, linkkejä, kuvia, videota, ääntä, jne. Merkittävä HTML:n ominaisuus on myös lomakkeet, jotka ovat usein yhteydessä tietokantoihin tai muihin taustajärjestelmiin, mahdollistaen näin interaktiivisen toiminnan kuten hakujen suorittamisen ja ostosten tekemisen.

Lisätietoja englanninkielellä virallisesta HTML 5:n spesifikaatiosta osoitteessa: https://www.w3.org/TR/2014/REC-html5-20141028/introduction.html#introduction


Mitä CSS -tyylisivut ovat?

CSS -tyylisivut määrittelevät sen, miltä dokumentit näyttävät kuvaruudulla, tulostettuna tai jopa ääneen lausuttuna. W3C on aktiivisesti kannustanut tyylisivujen käyttöä Wepissä aina World Wide Web Consortiumin perustamisesta, jo vuodesta 1994 lähtien. Style Activity -ryhmä on tuottanut lukuisia W3C:n suosituksia (CSS1, CSS2, CSS3, XPath, XSLT).

Liittämällä tyylisivun www-dokumenttiin (esim. HTML tai XML) sivujen tekijät ja lukijat voivat vaikuttaa dokumenttien ulkoasuun merkittävästi monipuolisemmin kuin pelkästään HTML-komentoja käyttämällä alustariippumattomuuden kärsimättä.

Sivun kehittäjän on helpoin tutustua tyylisivuihin käyttämällä nykyaikaisia selaimia kuten Firefox, Safari, Chrome, Opera, Edge ja Internet Explorer ja tutkia HTML sekä CSS koodia selaimen kehittäjätyökaluilla (developer tools). Tyylisivuista on tunnettu käytettävyysasiantuntija Jacob Nielsen sanonut mm. seuraavaa: “Hopefully, future Web innovations will emulate the example set by the Web Consortium in its work on CSS”.

Lisätietoja CSS:stä saa W3C:n kotisivuilta https://www.w3.org/Style/CSS/


JavaScript ja web-sivut

JavaScript on kevyt skriptikieli jota kutsutaan joskus myös leikillisesti webmasterin ohjelmointikieleksi.
JavaScriptiä käytetään pääosin web-sivuilla parantamassa sivuston käyttökokemusta. JavaScriptin avulla voi mm. manipuloida HTML-dokumentin rakennetta ja ulkoasua sekä tarkistaa HTML-lomakkeiden tietoja ennen kuin ne lähetetään palvelimelle.

Modernit selaimet voivat JavaScriptin avulla myös keskustella web-palvelimen kanssa. Tällaisia uudentyyppisiä sovelluksia kutsutaan yleisesti Ajax-sovelluksiksi. Ajax-sovelluksissa tiedon noutaminen ja lähettäminen tapahtuu asynkronisesti taustalla, jolloin selain on koko ajan aktiivisesti käytettävissä.

JavaScript -kielen kehitti aikoinaan Netscape -yhtiön palveluksessa ollut Brendan Eich, ja hän on edelleen kielen kehityksessä mukana. JavaScript -kielen kehitystä sekä dokumentaatiota voi seurata Mozilla -organisaation sivustolla https://developer.mozilla.org/fi/docs/Web/JavaScript.
